### Project Structure

    /gameproject
    |-- /bin          # Compiled executables
    |-- /build        # Intermediate build files (.o)
    |-- /include      # Header files (.h)
    |   |-- game.h
    |   `-- menu.h
    |-- /lib          # External libraries (e.g., Raylib)
    |-- /src          # Source code files (.c)
    |   |-- main.c
    |   |-- game.c
    |   `-- menu.c
    `-- Makefile
